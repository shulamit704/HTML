# Memory Game - HTML Project

This is a simple memory card game built using HTML, CSS, and JavaScript.

## About the Game

The goal of the game is to match pairs of cards by flipping them two at a time.
If the cards match, they stay open. If not, they flip back.

## Features

* Simple and clean design
* Card flipping interaction
* Matching logic implemented in JavaScript
* Easy to play

## Technologies Used

* JavaScript   
* HTML
* CSS

## How to Play

1. Open the `index.html` file in your browser
2. Click on cards to flip them
3. Try to find all matching pairs

## Project Structure

* `home.html` – main file
* `style.css` – design
* `the_game.js` – game logic
* `img/` – images
